# Expense Tracker CLI App
A simple command-line Python app to track your daily expenses.

## Features
- Add expenses with category and amount
- View total expenses
- Save data to a file

## How to Run
```bash
python expense_tracker.py
```

## Future Improvements
- View expenses by category
- Monthly summaries
- Export to CSV
